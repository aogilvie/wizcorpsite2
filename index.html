  • Methodology

    Exceptional page-load speed is achieved through multiple techniques. This theme comes with a PHP delivery system which assures the following tasks:

    • Automatic minifying and merging of javascript and CSS files.
    • Deliver gzipped content (JS, CSS, HTML and even font files).
    • Server-side caching of resulting minified and gzipped files.
    • Client-side caching rules based on name versioning.

    Requirements

    All you need is a simple PHP web hosting – nothing to configure, no technical skills required.

    The Project Source Features were engineered with performance and end-user experience in mind, but not only. The incredible simplicity of the HTML syntax allows almost anybody to customize and adapt the theme to each specific need, whether this is done manually or even through a visual editor.
