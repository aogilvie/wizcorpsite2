Performance

page-load speed

Techniques like CSS-sprites, asynchronous load and DOM events are intelligently used to improve page-load time and responsiveness.

Cross-Browser

and cross-device

Fully optimized for all browsers and mobile devices using progressive enhancement with graceful fallback. It's pixel perfect!

Bookmarkable URLs

and browser history

This enterprise-level website offers dynamic URL updates, meaning bookmarkable URLs and a fully functional browser history.

Flexible Layout

responsive design

Based on a lightweight 960 CSS grid and using advanced Media Queries, this website offers a responsive grid down to mobile. Go ahead, resize this page!

Smart CDN Fallback

combine cloud and independance

Providing CDN-based components results in performance improvement for end-users, and the great part is that there's still a local fallback in case of CDN failure.

the power of HTML5 standards

Making use of the newest W3C standards, HTML5 structure and content-specific tags, we offer not only top accessibility but also natural SEO-oriented enhancements.

  • Methodology

    Exceptional page-load speed is achieved through multiple techniques. This theme comes with a PHP delivery system which assures the following tasks:

    • Automatic minifying and merging of javascript and CSS files.
    • Deliver gzipped content (JS, CSS, HTML and even font files).
    • Server-side caching of resulting minified and gzipped files.
    • Client-side caching rules based on name versioning.

    Requirements

    All you need is a simple PHP web hosting – nothing to configure, no technical skills required.

    The Project Source Features were engineered with performance and end-user experience in mind, but not only. The incredible simplicity of the HTML syntax allows almost anybody to customize and adapt the theme to each specific need, whether this is done manually or even through a visual editor.
